## N. Key Features & Technologies Integration

### Application Features
1. **Building Permit Management**
   - Application submission and tracking
   - Status updates (Pending, Approved, Rejected, Resubmit)
   - FIFO (First In First Out) processing
   - Search and filter functionality

2. **User Authentication**
   - Google OAuth login
   - JWT-based sessions
   - Role-based access control (Applicants, Staff)

3. **Notifications**
   - Email notifications (Resend API)
   - In-app toast notifications (Sonner)
   - Staff notification bell
   - Applicant notification system

4. **File Management**
   - Attachment uploads to AWS S3
   - File preview and download
   - Secure presigned URLs

5. **Dashboard & Analytics**
   - Real-time data visualization (Recharts)
   - Application statistics
   - Performance metrics

6. **PDF Generation**
   - Permit document export
   - HTML to PDF conversion (jsPDF + html2canvas)

7. **Data Export**
   - CSV export (PapaParse)
   - Excel export (XLSX)

8. **UI/UX Features**
   - Dark/Light theme toggle
   - Responsive design
   - Smooth animations (Framer Motion)
   - Skeleton loaders
   - Error boundaries

## P. API Architecture

### API Protocol
- **Type**: TRPC (Type-safe RPC)
- **Transport**: HTTP REST-like with JSON
- **Authentication**: JWT tokens in secure cookies

### API Categories
1. **Authentication**
   - Login (via Google OAuth)
   - Logout
   - Session validation

2. **Applications**
   - Create application
   - Get applications (with FIFO order)
   - Search applications
   - Get by reference number
   - Update status
   - Delete application

3. **Activity Logs**
   - Log application activity
   - Get activity history

4. **Notifications**
   - Log notifications
   - Send emails
   - Trigger staff notifications

5. **System**
   - Health checks
   - System information
   - Configuration
